# Роль
Ты — Senior Frontend Developer и эксперт по конверсионному маркетингу в РФ. Твоя задача — создать высококонверсионный лендинг для клининговой компании.

# Технологический стек
- HTML5 (семантическая верстка)
- CSS3 (Flexbox, Grid, переменные, адаптивность)
- JavaScript (Vanilla ES6+, без тяжелых фреймворков)
- Интеграция: Telegram Bot API (для заявок)

# Контекст и География
- Регион: Россия (Москва/СПб/Регионы)
- Язык: Русский
- Валюта: Российский рубль (₽)
- Законодательство: соблюдение 152-ФЗ (Персональные данные)

# Структура Лендинга (По секциям)

1. **Header (Шапка)**
   - Логотип (текстовый "CleanPro").
   - Меню: Услуги, Цены, Отзывы, Контакты.
   - Контакты: Крупно телефон +7 (999) 000-00-00, иконки WhatsApp, Telegram, VK.
   - Кнопка: "Заказать звонок".

2. **Hero Section (Первый экран)**
   - Заголовок: "Профессиональная уборка квартир и офисов в [Город]".
   - Подзаголовок: "Чистота без хлопот. Выезд в день заказа. Гарантия качества."
   - Буллиты: Эко-средства, Опытные клинеры, Оплата после результата.
   - CTA Кнопка: "Рассчитать стоимость" (скролл к форме).
   - Фон: Качественное фото интерьера или процесса уборки.

3. **Trust Block (Цифры)**
   - 5+ лет на рынке.
   - 1000+ уборок выполнено.
   - 4.9 Рейтинг на Яндекс.Услугах.
   - Страховка ответственности.

4. **Services & Pricing (Услуги и Цены)**
   - Карточки услуг: Поддерживающая, Генеральная, После ремонта, Химчистка.
   - Цены в рублях (например: "от 2 500 ₽").
   - Чек-лист "Что входит в уборку" (аккордеон или список).

5. **Before/After (Галерея)**
   - Слайдер или сетка с фото "До" и "После".

6. **Lead Form (Форма захвата)**
   - Поля: Имя, Телефон (маска +7), Тип уборки, Площадь, Комментарий.
   - Чекбокс: "Согласен на обработку персональных данных" (обязательно для 152-ФЗ).
   - Кнопка: "Отправить заявку".
   - Логика: Отправка данных в Telegram бота через fetch API.

7. **Reviews (Отзывы)**
   - 3 карточки с именем, городом, текстом и 5 звездами.

8. **Footer (Подвал)**
   - Реквизиты (ИП/ООО, ИНН).
   - Политика конфиденциальности (ссылка).
   - Контакты снова.
   - Копирайт.

# Дизайн-система
- Цвета: Белый (#FFFFFF), Голубой (#E3F2FD), Акцентный синий (#2196F3), Текст (#333333).
- Шрифты: 'Inter' или 'Roboto' (подключить Google Fonts).
- Стиль: Минимализм, чистота, воздух между блоками.
- Адаптивность: Mobile First (корректное отображение на 320px+).

# Технические требования
1. **Валидация формы:** Проверка телефона (регулярное выражение РФ), обязательные поля.
2. **Маска телефона:** Реализовать на JS (формат +7 (___) ___-__-__).
3. **Отправка в Telegram:** 
   - Написать функцию `sendToTelegram(formData)`.
   - Использовать `fetch` к API Telegram.
   - Предусмотреть обработку ошибок (try/catch) и уведомление пользователя (alert/modal).
   - *Важно:* В коде оставить комментарии, где вставить BOT_TOKEN и CHAT_ID.
4. **SEO:** Meta tags (title, description), alt у изображений, h1-h6 иерархия.
5. **Производительность:** Минимум запросов, оптимизированный CSS/JS.
6. **152-ФЗ:** Галочка согласия должна быть активна по умолчанию или требовать клика (уточнить в коде комментарием).

Предоставь полный код в трех блоках:

2. `styles.css`
3. `script.js`
